A simple plugin to show amount of shares among popular networks on the edit page.

== Description ==

This plugins shows the social reach of your post/page in the admin panel while editing.

Social Reach displays number of likes and shares on Facebook (combined), shares on Twitter and +1's from Google Plus and a combined value of all as a total score.

== Installation ==

1. Upload `SocialReach.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You're good to go!
